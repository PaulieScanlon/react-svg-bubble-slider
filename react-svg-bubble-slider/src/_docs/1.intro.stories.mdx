SvgBubbleSlider is fun way to add reactions to your blog posts ğŸ˜ƒ ğŸ˜  ğŸ˜¥ and comes equipped with 14 built in reactions, but you don't have to use them all, see the <Link href='/?path=/docs/svgbubbleslider--icons'>icons prop</Link> to modify the icons list.

You can access the current <Link href='/?path=/docs/svgbubbleslider--reaction'>reaction</Link> via a render prop and use it to post to your back-end of choice.

Click, drag or use your keyboard arrows to explore all the reactions.

### Getting Started ğŸš€

You can install SvgBubbleSlider as a node module via `npm` or `yarn`

```sh
npm install react-svg-bubble-slider
```

<Divider />

### Usage ğŸ§°

To use SvgBubbleSlider import it and add it to your React component's render method

```javascript
// some-component.js/.tsx
import React from 'react'

import { SvgBubbleSlider } from 'react-svg-bubble-slider'

const SomeComponent = () => {
  return (
    <div>
      ...
      <SvgBubbleSlider />
    </div>
  )
}

export default SomeComponent
```

### SvgIcon ğŸ˜ƒ ğŸ˜  ğŸ˜¥

In addition to allowing users to leave a reaction you may wish to display how many reactions have been left. One way to do this is to use the `SvgIcon` and display the count underneath it. Also, checkout the <Link href='/?path=/docs/svgbubbleslider--icon-set'>iconSets</Link> prop to use an alternative set of icons
