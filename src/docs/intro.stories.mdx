# SvgBubbleSlider

A React version of Chris Gannon's [SVG Bubble Slider](https://codepen.io/chrisgannon/pen/GZNgLw/)

<SvgBubbleSlider>
  {({ reaction }) =>
    reaction && (
        <Button
          sx={{
            textTransform: 'capitalize',
          }}
        >
          {`${reaction}`}
        </Button>
    )
}

</SvgBubbleSlider>

### Install

<Divider />

```sh
npm install ...
```
